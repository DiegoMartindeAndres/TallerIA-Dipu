## 🎯 ¿Qué es una Gema?

### Antes (Prompting Tradicional)
```
Usuario: Necesito ayuda con normativa...
IA: (responde de forma genérica)
Usuario: No, es que en nuestro caso...
IA: (olvida el contexto previo a cada prompt)
```

### Ahora (Con una Gema)
```
Usuario: Necesito analizar esta normativa
Gema: (ya conoce todas las normativas de tu administración)
Gema: (sabe cómo trabajáis)
Gema: (da respuestas contextualizadas)
```

Una **Gema es un contenedor de conocimiento** que empaqueta:
- 📚 Instrucciones especializadas
- 🧠 Contexto y conocimiento de dominio
- 🎯 Ejemplos específicos
- ⚙️ Reglas y restricciones

## 💡 Ventajas de las Gemas

### vs Prompting tradicional

| Aspecto | Prompting | Gemas |
|--------|-----------|-------|
| **Memoria** | Cada pregunta es nueva | Mantiene contexto |
| **Consistencia** | Puede variar mucho | Siempre responde igual |
| **Especialización** | Genérica | Muy especializada |
| **Reutilización** | Tienes que recordar el prompt | Solo acciona la Gema |
| **Escalabilidad** | Limitada | Puedes tener 20+ |
| **Compartir** | Complicado | Copias y compartes |

## 💬 Preguntas Frecuentes

### "¿Qué diferencia hay con una carpeta de prompts?"
Las Gemas tienen MEMORIA e IDENTIDAD. Recuerdan contexto entre preguntas. Los prompts son aislados.

### "¿Cuántas Gemas puedo crear?"
Ilimitadas. Muchas personas tienen 20-30 Gemas especializadas.

### "¿Puedo compartir mis Gemas?"
Sí. Las puedes copiar, enviar por email, compartir en Drive.

### "¿Qué pasa si mejoro una Gema?"
La siguiente versión será mejor. Es un proceso iterativo.

### "¿Cuánto tiempo tarda crear una Gema?"
De 30 minutos a 2 horas, dependiendo de la complejidad.
