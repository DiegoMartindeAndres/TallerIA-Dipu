# 🎨 Generación de Imágenes: Nana Banana Pro

## 📌 Objetivo

Aprender a generar imágenes profesionales con IA para carteles, posters y materiales promocionales usando **Nana Banana Pro**.

En esta sección descubriremos cómo:
- Transformar ideas en imágenes visuales
- Crear carteles para cursos y eventos
- Iterar rápidamente sobre diseños
- Optimizar prompts para obtener resultados consistentes

---

## 🎯 ¿Qué es Nana Banana Pro?

**Nana Banana Pro** es un generador de imágenes basado en IA que:

✅ Crea imágenes de alta calidad a partir de descripciones de texto
✅ Mantiene coherencia visual en múltiples generaciones
✅ Permite ajustar estilos, colores y composición
✅ Es accesible y fácil de usar (perfecto para no diseñadores)
