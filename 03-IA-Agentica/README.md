## 🎯 ¿Qué es un Agente?

### La Evolución

```
Chatbot:      "¿Cuál es la capital de Francia?" → "París"

Agente:       "Necesito generar un informe de subvenciones"
              → Planifica pasos
              → Accede a bases de datos
              → Redacta documento
              → Valida formatos
              → Entrega resultado
```

### Diferencias Clave

| Característica | Chatbot | Agente |
|---|---|---|
| **Iniciativa** | Pasivo (espera preguntas) | Activo (ejecuta tasks) |
| **Planificación** | No | Sí |
| **Herramientas** | Ninguna | Muchas |
| **Memoria** | Conversación actual | Persistente |
| **Autonomía** | Nula | Limitada pero real |
| **Tiempo** | Segundos | Minutos/horas |
